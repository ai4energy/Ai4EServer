Ai4EServer API

MeathodAPIInformation
GET/API列表与示例
GET/health测试API状态
POST/job直接返回POST提交内容
POST/api/modeljson计算ModelJson格式的JSON文件
POST/api/getResult获得最新一次计算的结果

Examples

API: /health

using HTTP
res = HTTP.get("http://127.0.0.1:8081/health")
@show String(res.body)
String(res.body) = "{\"Julia-API\":\"healthy!\"}"

API: /job

using HTTP
res = HTTP.request("POST", "http://localhost:8081/job", [], """{"name":"Hello"}""")
String(res.body)
String(res.body) = "{\"Your Message\":\"{\\\"name\\\":\\\"Hello\\\"}\"}"

API: /api/commonjson

using HTTP
str = """
{
    "name": "Name",
    "pkgs": [
        "ModelingToolkit",
        "DifferentialEquations"
    ],
    "variables": [
        "x(t) = 1.0",
        "y(t) = 1.0",
        "z(t) = 2.0"
    ],
    "parameters": [
        "σ = 1.0",
        "ρ = 3.0",
        "β = 5.0" 
    ],
    "equations": [
        "der(x) = σ*(y - x)",
        "der(y) = x*(ρ - z) - y",
        "der(z) = x*y - β*z"
    ],
    "u0": [
        "x => 1.0",
        "y => 2.0",
        "z => 3.0"
    ],
    "timespan": [0,1],
    "solver": "Rosenbrock23"
}
"""
res = HTTP.request("POST", "http://127.0.0.1:8081/api/commonjson"; body = str)
String(res.body)
String(res.body) = "{\"State\":\"Success\"}"
